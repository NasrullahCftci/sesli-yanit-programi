# Voice Reader App

Metni sesli olarak okuyan Flutter uygulaması.

## Özellikler

- Metin girişi için kullanıcı arayüzü
- Text-to-Speech (TTS) ile sesli okuma
- Çoklu platform desteği (Web, Android, iOS, Windows, Linux, macOS)
- Responsive tasarım

## Kurulum

```bash
flutter pub get
```

## Çalıştırma

Web için:
```bash
flutter run -d chrome
```

Windows için:
```bash
flutter run -d windows
```

Android için:
```bash
flutter run -d android
```
